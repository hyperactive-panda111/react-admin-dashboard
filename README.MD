## Introduction
React-based CRM dashboard featuring comprehensive authentication, antd charts, sales management, and a fully operational kanban board with live updates for real-time actions across all devices.

Tech Stack

- React.js
- TypeScript
- GraphQL
- Ant Design
- Refine
- Codegen
- Vite
